# FitTracker PRO

App web per il monitoraggio di allenamento, dieta e progressi fisici.

## Funzionalità
- Multi-utente con registrazione e login
- Piano dieta adattivo con gestione intolleranze
- Schede allenamento personalizzabili
- AI Coach integrato (Claude)
- Analisi progressi e grafici
- Funziona da telefono come una vera app
